# Client
To run this project, follow the below steps:

## Install dependencies
```
$ open the CMD from Client/ folder as root directory
$ npm install
```
## Build files
```
$ open the CMD from Client/ folder as root directory
$ npm run build
```
## Host 
```
$ open the CMD from Client/ folder as root directory
$ npm run host
```
open the link: http://localhost:3000
